# Hackaton Teatro 4D

> Gestionale Rete 4D Teatro

## Utilizzo

installa il venv:

```sh
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

installo l'app:

```sh
python manage.py startapp scambio
```
